Introduction
The search function is a powerful tool within the Saanich Parks & Rec App, enabling you to quickly sort through the thousands of programs we have to offer.

Within the search page you are able to input a series of parameters to refine your search, Any and all of the parameters are allowed to be left blank and will be ignored when searching.

Search Options
Activity
Matches your input keyword against all categories and program names to return a result. This parameter heavily filters the results of your search so consider inputting this field if you wish for faster searches!
Age
The age in which you want your program to be available to. The granularity of this is set to Years by default but may be set to Months if you wish to search for programs designed for infants.
Instructor
The instructor which teaches a course.
Municipality
The district limitation
Days of the Week
By default the search will match programs which are offered on all the days of the week, you may include as many days as you wish or as little as one.
Days of the Month
By default the search will match all programs offered at any time throughout the year. This can be refined to search for the availability of a program or activity at specific times throughout the year.

Additional Queries
Additional queries can be made by selecting the 'Add Query' button, This functionality enables you to combine two or more search results into a single request.

A new query will exist independantly from your others, meaning any parameters set within one query will not be carried over to another.
